# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
|  7.6.1  | :white_check_mark: |
| < 7.6.0 | :x:                |

## Reporting a Vulnerability

If you encounter a security vulnerability in I-Nex, you can tell us about it by
[opening an issue](https://github.com/i-nex/I-Nex/issues/new) here on GitHub.
